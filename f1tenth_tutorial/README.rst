F1tenth
=======

.. image:: robot.png


Usage
-----

To launch the F1/10 racecar in a Gazebo simulation:

.. code:: command

   $ roslaunch f1tenth-sim simulator.launch


To launch the simulation and `move_base`:

.. code:: command

   $ roslaunch f1tenth-sim navigation.launch
