Usage
-----

To launch the AutoRally platform inside a simulated world using Gazebo:

.. code::

   # source /ros_ws/src/autorally/autorally_util/setupEnvLocal.sh
   # roslaunch autorally_gazebo autoRallyTrackGazeboSim.launch
